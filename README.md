# MorCSA

AP Computer Science A learning game site focused on a roughly 10-hour score-5 prep path.

## Scripts

- `npm run dev` starts the local esbuild dev server on `http://127.0.0.1:4173`
- `npm run build` typechecks the app and writes the production files to `dist/`
- `npm run preview` serves the built `dist/` folder
